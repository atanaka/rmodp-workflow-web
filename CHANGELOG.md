# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-01

### Added
- Initial public release of `rmodp-workflow-web` skill suite
- 7 skills: orchestrator + 6 viewpoint sub-skills
- Bilingual support (Japanese / English) with auto-detection
- PlantUML diagram generation (PNG) for all viewpoints
- Resume support via `workflow-status.json`
- Phase splitting (Phase A / Phase B+C in separate sessions)
- Clarifying questions (3–5 per step) for specification refinement
- Progress templates in Japanese and English
- Example: Library Lending Management System scenario
